# Device Information

Device information and parsers for IO pad libraries.

## Components

- `IO_device_info_T28_parser.py` - Parser for 28nm IO device information
- `IO_device_info_T28.json` - JSON device database
- `IO_device_info_T28.txt` - Text device database
- `IO_device_pin_rules_T28.json` - Pin placement rules

## Usage

Provides device templates and pin rules for schematic and layout generation.
