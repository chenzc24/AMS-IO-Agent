## New Architecture

**Before (Smolagents):**
```
User → CLI/API → Agent Factory → CodeAgent → @tool decorated tools → Core logic
```

**After (Claude Code):**
```
User → Claude Code → MCP Server → MCP Tools → Core logic (direct calls)
```
