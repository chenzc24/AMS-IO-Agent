# Application Core Modules

Core application modules for IC design automation.

## Modules

- **cdac/** - CDAC (Capacitor Digital-to-Analog Converter) design agent
- **layout/** - Layout generation and visualization
- **schematic/** - Schematic generation
- **intent_graph/** - Intent graph validation and processing
- **utils/** - Utility functions and agent factories

## Usage

These modules are used by the main agent system to generate IC designs. Each module handles a specific aspect of the design workflow.
