# CDAC Agent

Intelligent CDAC (Capacitor Digital-to-Analog Converter) design agent that uses AI reasoning to determine array structures.

## Features

- Excel specification observation and analysis
- Unit capacitor design
- Dummy capacitor generation
- Array assembly automation
- No hard-coded rules - AI-driven reasoning

## Main Component

- `cdac_agent.py` - Main CDAC agent implementation with observation tools
