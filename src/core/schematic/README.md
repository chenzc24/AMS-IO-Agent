# Schematic Generation

Schematic generation module for IO rings and other IC components.

## Components

- `schematic_generator_T28.py` - Generate SKILL code for 28nm schematics
- `schematic_generator_T180.py` - Generate SKILL code for 180nm schematics

## Features

- Device template management
- Pin placement and routing
- Voltage domain handling
- SKILL code generation

## Usage

Used by the IO ring generator tool to create schematic SKILL code from configuration files.
