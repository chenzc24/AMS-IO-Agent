# Agent Generated Tools

Python tools automatically generated by the AI agent during conversations.

## Purpose

This directory contains reusable Python tools created by the agent to avoid code repetition. These tools are generated on-demand and persist across sessions.

## Files

- `read_skill_file.py` - Read SKILL file content
- `create_text_file.py` - Create text files
- `save_json_file.py` - Save JSON files
- `save_json_with_directory.py` - Save JSON with directory creation
- `fibonacci_sequence.py` - Fibonacci sequence generator

## Usage

These tools are automatically loaded and available to the agent. They can be created, updated, or deleted during agent conversations.
